# Frameworkiem?

<Note>
  React jest wyjątkowo otwartą biblioteką. Nie narzuca żadnych konwencji, nie
  wymusza żadnego stylu programowania. Można z niego korzystać na wiele
  sposobów, a nawet tworzyć własne frameworki, które będą go wykorzystywać.
</Note>

<Note>
  Właśnie ta otwartość jest jedną z głównych przyczyn, dla których React jest
  tak popularny. Dzięki temu, że nie narzuca żadnych konwencji, można go używać
  w dowolny sposób, który jest najbardziej odpowiedni dla danego projektu.
</Note>
